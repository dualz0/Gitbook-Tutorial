用 Git 做版本管理

用 Git 做版本管理

  1. 在 Github 创建好仓库
  2. 在 bookname 目录下执行 git init初始化本地仓库
  3. git remote add origin git@server-name:path/repo-name.git关联远程仓库。
  4. 关联后，使用命令git push -u origin master第一次推送master分支的所有内容。
